# Electron 番茄钟应用

## 项目简介

这是一个基于 Electron 的番茄钟应用程序，采用 HTML、CSS 和 JavaScript 实现。它帮助用户通过番茄工作法提高专注力和工作效率，支持工作和休息时间的切换。

## 功能说明

- **计时功能**：25分钟工作时间 + 5分钟休息时间的循环
- **开始/暂停**：控制计时器的启动和暂停
- **跳过**：立即跳转到下一个阶段（工作/休息）
- **重置**：将计时器重置到初始状态
- **界面提示**：显示当前状态（工作中/休息中/已暂停）

## 环境要求

- Node.js (version 14 或更高版本)
- npm (version 6 或更高版本)
- Electron (已通过 package.json 配置)

## 安装依赖

```bash
npm install
```

## 运行方式

```bash
npm start
```

或者

```bash
electron .
```
